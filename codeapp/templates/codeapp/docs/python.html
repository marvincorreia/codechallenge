Writing to standard output

Use the print function to write on standard output

Simple print:

Try sample

Escaping new lines:

Read from standard input

Use the input function to read from standard input, note that the entry must be defined before running the code

name = input()
print(f"Welcome {name}")
Try sample

Reading multiple entries:

x = int(input())
y = int(input())
print(f"{x} + {y} = {x+y}")
