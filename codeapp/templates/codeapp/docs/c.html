Read from standard input

To read entries you have several options such as scanf, gets, fgets, getline and more functions to read from standard input, note that the entry must be defined before running the code.

scanf reads the entry until it finds a space or line break

#include <stdio.h>

int main(){
    char name[50];
    scanf("%s",name);
    printf("Hello %s",name);
}

gets read a entire buffer so its deprecated and not recommended, use fgets instead

#include <stdio.h>

int main(){
    char name[50];
    gets(name);
    printf("Hello %s",name);
}

fgets reads a entire line, its a good practice

#include <stdio.h>

int main(){
    char name[50];
    fgets(name,sizeof(name),stdin);
    printf("Hello %s",name);
}
